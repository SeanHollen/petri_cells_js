Petri Cells

An implementation of the sim from this paper which explores how self-replicating programs can emerge spontaneously from cellular autonoma. Runaway "life" only emerges occasionally, so run many times (and get lucky).
Every epoch, a given cell will be concatenated with one of its neighbors, the combined program will be run (as per the interpreter below), and the resultant will be split to create new cells.
I learned about it from this Sabine Hossenfelder video. Read those sources for more detailed descriptions of what is happening.
